About Sell Mobiles

What we do

Sell Mobiles is a site where users can find second hand mobiles on sale and can directly contact the sellers to buy the mobile. The users can also post mobiles to be sold on the site. This site gives the facility to chat directly with the seller so as to negotiate accordingly. This easy to handle site makes the process easy and safe for the seller as well as the buyer. The admin plays a role in administrating the site data, functionality and flow. The admin has the authority to edit or change the features on the site. The user can visit our site and search various mobiles based upon the locations as well as the brands. When a seller would like to post a mobile , he/she needs to sign in to get redirected to a 3 step form where he/she can upload the details of the mobile to be posted. The awesome UI of the site makes it easy for the user to access information from the site as well as to sell the second hand mobile directly to any other user with a simple step i.e. The chat

What we have

The Chat with seller option helps the user to get connected with the seller and chat with him for buying his phone open for sale. The like option for phone is given for the user to like the respective mobiles he feels good and can add them to his wishlist so that he can view them later. The history feature helps the user to track his particular details on this site, the mobiles he'd sold and the mobiles he'd bought. the mobiles uploaded feature helps the user to view all his uploaded mobiles on the website at one place so that he can find and edit the respective mobile
